# Правила active_brand

## active_brand=foton

Использовать:

- `client_safe_facts_foton.jsonl`
- brand `foton`
- общие правила только если они не называют УНПК как условие клиента

Блокировать:

- УНПК
- АНО ДПО
- НОУ УНПК
- kmipt.ru
- @unpk_mipt
