# Good Manager Answers Candidates

Это не база фактов. Это очередь стилевых приемов, которые можно использовать после ручного отбора.

Важно: paid-proxy звонки в локальном snapshot перекошены в одного менеджера. Перед утверждением style playbook нужен ручной баланс источников, иначе “gold-стиль” может стать стилем одного человека, а не команды.

## Нельзя

- Копировать ответы менеджеров дословно.
- Считать историческую формулировку актуальным фактом.
- Вносить цены, скидки, даты, места, сроки и платежные условия без первичного источника.
- Использовать ответы с персональными обещаниями или индивидуальными условиями.
