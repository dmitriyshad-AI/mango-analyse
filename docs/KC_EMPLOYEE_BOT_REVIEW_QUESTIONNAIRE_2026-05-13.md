# Опросник для сотрудников: что бот может говорить сам

Дата: 2026-05-13
Назначение: проверить правила автономности бота и 50 примеров ответов.

## Как проверять

Для каждого пункта поставьте решение:

- `утвердить` — можно использовать в черновиках клиентских ответов;
- `исправить` — нужно заменить формулировку;
- `запретить` — не использовать;
- `только менеджер` — бот не отвечает по сути;
- `нужен источник` — нужна ссылка на документ/сайт/регламент.

## Что бот может делать сам

| Тема | Предложение | Решение сотрудника |
|---|---|---|

| Ссылки | Давать ссылки на публичные страницы курсов, контактов и акций |  |
| Пробное занятие | Не обещать, а передавать менеджеру после уточнения класса/предмета |  |

5. Решение по пробным занятиям и рассрочке.
