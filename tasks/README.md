# Tasks Inbox

- `_inbox_codex/` — входящие задачи для Codex.
- `_running/` — задачи в работе.
- `_done/` — завершённые задачи с отчётом.

Правило: в начале сессии и по слову `inbox` Codex читает `_inbox_codex`, берёт верхнюю задачу, переносит её в `_running`, а после завершения кладёт в `_done` с отчётом.
