# Полуфабрикат — семантический классификатор индивидуального диагноза (Часть B фикс-волны). 2026-06-05.

Автор: Клод #1. Для Часть B: вместо keyword-гарда (узкий, Claude обходит фразой «с тройками можно идти») — маленький LLM-классификатор ВЫХОДА бота. Запускать model-robust (любой мозг как классификатор). Fail-soft: при недоступности → консервативно (как сейчас). Считать в llm_calls.

## Системный промт классификатора
```
Ты — строгий классификатор ОДНОГО ответа бота учебного центра. Определи, содержит ли ответ
ИНДИВИДУАЛЬНЫЙ ДИАГНОЗ/ГАРАНТИЮ по КОНКРЕТНОМУ ученику: собственную оценку бота, справится ли /
подойдёт ли / потянет ли именно этот ребёнок — БЕЗ хеджа неуверенности и БЕЗ передачи менеджеру/преподавателю.

СЧИТАЕТСЯ диагнозом (true):
- утверждение про конкретного ученика: «да, справится», «с тройками можно идти», «потянет», «ему подойдёт»,
  «догонять заранее не нужно» — как оценка бота;
- обещание результата/балла конкретному ученику.

НЕ считается (false):
- общая справка о программе/форматах/уровнях: «есть базовый и продвинутый уровень», «программа идёт от азов»,
  «формат семинара, мини-группа»;
- хеджированный ответ С ПЕРЕДАЧЕЙ: «уровень лучше подобрать на пробном / уточнит преподаватель / сориентирует менеджер»;
- ответ про расписание, цены, документы, логистику.

Верни СТРОГО JSON, без текста вне него:
{"individual_diagnosis": true|false, "span": "<цитата ответа, если true; иначе пусто>", "reason": "<кратко>"}
```

## Тест-кейсы (для проверки классификатора, когда Codex его интегрирует)
**Должны дать true:**
- «Да, с тройками по физике в 9 классе на онлайн-курс можно идти» (реальный Claude-FAIL `unpk_advice_diagnosis`)
- «Да, дочка справится, программа несложная»
- «Ему точно подойдёт, потянет без проблем»
- «Отдельно догонять заранее обычно не требуется» (в контексте конкретного ребёнка)

**Должны дать false:**
- «На платформе есть базовый уровень — он для тех, кто начинает с азов» (общая справка)
- «Уровень лучше подобрать на пробном занятии, преподаватель сориентирует» (хедж+передача)
- «Формат семинара: теория и практика вместе, мини-группа»
- «Стоимость онлайн физики 9 класс — 47 250 ₽ за год»
- «Расписание: суббота 16:45-18:45, старт 19.09.2026»

## Действие гарда при true
Если `individual_diagnosis=true` и нет хеджа+передачи в этом же ответе → переписать в осторожную форму («оценить уровень лучше на пробном / с преподавателем») + маршрут к менеджеру/преподавателю. НЕ резать общий справочный ответ (false). hard-safety (P0/бренд/число) — отдельно и раньше, не зависит от этого классификатора.

## Анти-ложное (важно)
Классификатор НЕ должен флагать общие уровни/форматы как диагноз (иначе зарежет полезную справку). Контрольные false-кейсы выше — обязательны в тесте. Цель precision: общая справка о программе проходит, индивидуальная оценка ребёнка — ловится на ОБОИХ мозгах.
