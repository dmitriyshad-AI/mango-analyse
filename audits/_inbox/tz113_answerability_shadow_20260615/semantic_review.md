# Semantic review

## Проверка смысла

Изменение не меняет клиентский ответ и не включает новую самостоятельность бота. Оно только собирает диагностический слой: могла ли модель, по собственному мнению, ответить сама, каких фактов ей не хватило и какой слой затем понизил маршрут.

## Что это даст

- Можно измерить лишние уходы к менеджеру без риска для клиентов.
- Можно сравнить мнение модели с текущими детерминированными гейтами.
- Можно готовить следующий этап миграции ответуемости на модель, не ломая текущий безопасный контур.

## Вердикт

semantic_pass для теневого диагностического слоя. Для использования как управляющей логики понадобится отдельный замер и новое ТЗ.
