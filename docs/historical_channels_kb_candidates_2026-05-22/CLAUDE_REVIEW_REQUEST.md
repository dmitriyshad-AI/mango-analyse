# Claude Review Request

Проверь пакет `docs/historical_channels_kb_candidates_2026-05-22/` как независимый смысловой ревьюер.

## Контекст

Задача пакета: read-only анализ исторических каналов для подготовки кандидатов улучшения базы знаний и будущих тестов.

Источники: Telegram, email, звонки, общий каталог вопросов, текущая KB v6.3.

Пакет не должен:

- утверждать факты;
- импортировать что-либо в KB;
- разрешать автоответы клиентам;
- раскрывать персональные данные;
- писать во внешние системы.

## Проверь

1. Не выдает ли пакет кандидаты за утвержденные факты?
2. Нет ли скрытого разрешения на автономные ответы из истории?
3. Достаточно ли жестко отделены P0, brand-risk, payment/status и attachments?
4. Есть ли риск ПДн или сырых фрагментов?
5. Полезен ли пакет для РОПа и следующего ТЗ?
6. Что обязательно исправить до финального ответа Дмитрию?

## Ожидаемый формат

- Verdict.
- Blocking findings.
- Non-blocking findings.
- Missing tests/gates.
- Final recommendation.
