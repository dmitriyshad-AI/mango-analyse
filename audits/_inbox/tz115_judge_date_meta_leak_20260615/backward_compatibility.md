# Backward compatibility

- API публичных функций не менялся.
- Regex парсинга текста бота не менялся.
- `_sanitize_humanity_meta_text` не менялся.
- `snapshot_number_index()` сохраняет прежние текстовые индексы и добавляет только узкий индекс окна из `before_...`.
- Поведение модели и клиентских ответов не менялось, кроме блокировки новой мета-утечки выходным гейтом.
