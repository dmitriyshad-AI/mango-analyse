# Опросник для обогащения базы знаний Telegram-бота

Дата: 2026-05-20  
Назначение: закрыть недостающие факты для P1-ответов бота после MEGA smoke v3.  
Как заполнять: отвечать коротко, но конкретно. Если точного ответа нет — писать `нет точного ответа`, `только менеджер`, `нужно проверить у РОПа` или `устарело`.

## 0. Общие правила заполнения

Для каждого ответа желательно указать:

- бренд: `Фотон`, `УНПК МФТИ` или `оба бренда`;
- можно ли говорить клиенту напрямую;
- нужно ли показывать только менеджеру;
- источник истины: сайт, Google Docs, документ, РОП, бухгалтерия, Tallanto, AMO;
- как часто обновлять факт;
- кто владелец факта.

Важно:

- Фотон и УНПК нельзя смешивать в одном клиентском ответе.
- Если факт верен для обоих брендов, всё равно лучше записать его двумя отдельными строками: отдельно для Фотона, отдельно для УНПК.
- Если факт требует проверки актуальности, бот не должен говорить его как точный.
