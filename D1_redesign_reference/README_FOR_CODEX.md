## Модули (reference/)
- `dialogue_understanding.py` — [0] детерминированный P0 пре-гейт + [1] LLM #1 → **контракт-ПЛАН**:
  `AnswerContract{ subquestions[{text,answerable,needed_fact_keys,next_step}], known_slots{name:Slot{value,source}}, client_state, denied_topics, ... }`.
  Слоты БЕЗ источника не утверждаются (`assertable_slots()` отдаёт только sourced). Модель называет КЛЮЧИ, не значения.
- `fact_provider.py` — [2] ретривал значений по ключам ТОЛЬКО из бренд-скоупного склада.
- `output_verifier.py` — [4] ЖЁСТКИЙ детерминированный verifier: brand_leak / fact_grounding(числа) / forbidden_scope / meta_leak / ai_disclosure / p0.
- `quality_layer.py` — [5] `check_claim_faithfulness` (ЖЁСТКО, LLM: не-числовые утверждения «по будням» ∈ facts/слова клиента?),
  [6] `form_check` (МЯГКО: штамп/повтор/нет шага/канцелярит → триггер тепла), `warmth_rewrite` (X2, меняет форму).
- `pipeline.py` — оркестратор полного потока + `Toggles{enforce_slot_evidence, semantic_faithfulness, form_warmth}` (изоляция компонентов при отладке).
- `tests/test_pipeline.py` — 11 тестов: контракт-план, слот-без-источника не утверждается, не-числовая выдумка ловится семантикой (репар/фоллбэк),
  тумблер семантики, form-check→тепло, тепло ре-верифицируется (ломающее факты тепло отвергается), бренд-утечка→фоллбэк, P0, manager_only.

## Поток
P0 пре-гейт → понять (контракт-план + client_state) → ретривал значений → черновик (свободно: client_state + стилевые примеры + ТОЛЬКО sourced-слоты + facts) →
ЖЁСТКИЙ verifier → СЕМАНТИЧЕСКАЯ верность → [fail] ремонт ≤2 / безопасный fallback → МЯГКАЯ форма → X2-тепло (РЕ-проверка жёстких) → финал.

## Инварианты безопасности (НЕ ронять при переносе)
1. Контракт называет КЛЮЧИ, значения — только из склада. 2. Слот без источника не утверждается (в черновик идут только sourced).
3. P0 детерминированный пре-гейт обязателен. 4. ЖЁСТКИЙ verifier + СЕМАНТИЧЕСКАЯ верность одновременно со свободой генерации.
5. Бренд-скоуп жёсткий. 6. Нет факта → узкий честный хендофф, не сосед. 7. Выход X2-тепла ОБЯЗАТЕЛЬНО ре-верифицируется; ломает факты → откат на исходный.
