# Опросник по коммерческим и программным фактам для базы знаний

Дата: 2026-05-13
Назначение: быстро проверить у РОПа и сотрудников, какие факты можно разрешить для клиентских черновиков бота.

## Как заполнять

В колонке “Решение” нужно поставить одно из значений:

- `утвердить для клиента`;
- `только черновик менеджеру`;
- `только менеджер отвечает`;
- `исправить формулировку`;
- `устарело`.
