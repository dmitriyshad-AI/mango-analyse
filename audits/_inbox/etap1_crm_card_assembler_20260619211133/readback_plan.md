# Readback plan

Для текущего блока readback из AMO не выполняется: live/writeback запрещены.

План для Этапа 2 перед записью:

1. Получить approved немаскированный AMO join-key из read-only источника.
2. Перед каждой записью снять snapshot целевых AI-полей и ручной `История общения`.
3. Проверить allowlist нижнего helper: только целевые AI-поля, без телефона/ФИО/email/manual history.
4. После writeback выполнить readback только записанных AI-полей.
5. Сверить, что ручные поля не изменились.
