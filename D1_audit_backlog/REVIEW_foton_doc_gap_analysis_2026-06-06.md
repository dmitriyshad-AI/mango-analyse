# Gap-анализ: «Вопросы по ФОТОНу и с чем их едят.docx» vs база v6.5 — 2026-06-06

**Документ:** памятка менеджеров за ПРОШЛЫЙ учебный год (Сретенка/Скорняжный = старые адреса, акции до 15.12.2025, расписания 2025/26). Текст ~24K знаков. Всё устаревшее отфильтровано, ниже только живое.

## A. ПРОБЕЛЫ базы (кандидаты в client-safe факты — после подтверждения Дмитрия)

1. **Описание уровней очных групп** — «Продвинутые vs Базовые: разница в темпе, сложности задач и времени на повторение; учебные планы почти не различаются. Олимп-группы помечены отдельно. Отдельных групп „только Физтех“ очно НЕТ». В базе уровни описаны только для онлайн-платформы. ⚠ Это ровно факт, которого не хватило в hp_diagnosis (бот выдумал «ОГЭ = для выравнивания») — закрыл бы целый класс производных.
2. **К онлайн-занятию вместо очного подключиться нельзя** (исключение: пропуск 4+ занятий подряд по болезни). В базе нет.
3. **Отработка пропуска — общий случай**: «прийти отработать в другой группе можно (по времени/уровню), тема может отличаться». В базе отработка привязана ТОЛЬКО к справке 095У/олимпиаде — документ шире. Противоречие/расширение.
4. **Конспекты: ПРОТИВОРЕЧИЕ с базой.** Док (жирно): «У нас НЕТ конспектов занятий, никогда не было» (альтернативы: чат препода — домашки/фото доски, методички на части курсов, попросить одноклассника). А факт базы про формат говорит «…самостоятельная работа, конспекты» — клиент может понять как «выдают конспекты». Надо развести формулировки.
5. **Домашнее задание можно выслать** (из чата с преподавателем) — в базе нет.
6. **Заморозка обучения / гибкий график** (при личных обстоятельствах) — в базе нет (grep пуст). Если политика официальная — сильный retention-факт.
7. **Бесплатное пробное занятие в другой группе при сомнениях** — как retention-предложение; в базе free-trial есть фрагментами, этой политики нет.
8. **Индивидуальные занятия — пакеты**: база знает только «45 мин — 3 900 ₽»; док добавляет 90 мин 7 900→6 900 (-12%) и пакет 5 занятий со скидкой до 41% (вёрстка цен кривая: «39 000 → 23 000 или 27 000»). Плюс клиентский процесс подбора (анкета → 2-3 кандидатуры → MTS-Link → оплата за 5, пробное 1 занятие).
9. **Результаты вступительного теста — в течение недели, в той же форме**; на первом очном занятии возможна устная диагностика, решения взять с собой. База знает про тест+анкету, но не сроки/механику.

## B. НЕ вносить (устарело/конфликтует — проверено)

- Адреса Сретенка/Скорняжный, расписания и ФИО преподавателей (прошлый год; ФИО клиенту не даём по политике).
- Онлайн-цены 24 800/17 550/28 100 и акция «до 15 декабря 2025» — прошлогодние, конфликтуют с актуальными.
- «Два предмета по цене одного (9-11)» — конфликтует с текущей политикой «второй предмет: Фотон 30% / УНПК 20%». Не вносить без явного решения.
- **М9/М11 — база ПРАВА**: тарифная сетка дока (18 900/47 250/59 500/94 500) — это ровно те «устаревшие края цен», что в базе уже помечены «не использовать»; статус discontinued подтверждается.
- Юрлица-роутинг оплат (кто платит на ЦРДО/АНО/НОУ) — internal_only, и датировано 01.12.25.

## C. Внутреннее золото (НЕ client-safe, но использовать)

- **Регламент отказов** (чек-лист + мини-скрипты по причинам: уровень→перевод, преподаватель→смена, время→другой формат, сомнения→бесплатное пробное, места→резерв, личное→заморозка) — прямой материал для ТОН-блока и A2-retention бота (мягкие случаи ДО P0), и для manager_only подсказок.
- Характеристики преподавателей («может заинтересовать маленьких физикой», «готовит 100-балльников») — manager_only рекомендации, клиенту регалии без ФИО.
- Приоритет записи: время → уровень/входные знания → наполненность (меньшие группы первыми) — selling-логика для A2.
