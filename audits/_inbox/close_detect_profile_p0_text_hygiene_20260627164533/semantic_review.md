# Semantic review

- Бренды: изменения не добавляют новых брендовых фактов и не меняют brand guard.
- Цены/даты/условия: изменения не добавляют коммерческих фактов; P0-гигиена, наоборот, удаляет обещания исхода возврата и продающие хвосты.
- P0: маршрут P0 не ослаблен. Smoke: `хочу снять ребёнка с кружка` -> `refund`, `верните деньги` -> `refund`, `снять стресс ребёнку` -> не P0.
- ПДн/внутреннее: в metadata P0-гигиены не сохраняется исходный опасный текст; пишется только факт применения и тип.
- Остаточный semantic-риск: включённый closing-fix должен пройти внешний регрейд на реальных closing/follow-up кейсах; P0-гигиена не включается без отдельного semantic pass.
