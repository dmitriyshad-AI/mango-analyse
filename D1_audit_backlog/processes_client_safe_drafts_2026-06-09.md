## 4. Тестирование и группы

### 4a Как проходит тест / где взять
**Фотон:** 1) Перед началом ребёнок проходит небольшое вступительное тестирование — определяет уровень, помогает подобрать группу. 2) Ссылку на тест и анкету присылаем при записи (на почту). 3) Тест включает открытые задания, проверяются вручную. Источник: [KB placement_and_first_feedback] + [из диалогов — кандидат].
**УНПК:** 1) Перед стартом (а для смен — перед сменой) проводится вступительное тестирование по уровню и классу. 2) Тест присылаем на почту; для выездной школы обычно за 2–3 недели до смены, онлайн. 3) Отбора детей нет — тест только для распределения по группам. Источник: [KB lvsh_placement_test] + [из диалогов — кандидат].
**Semantic-note:** бренд ок. Конкретные ссылки на тест не выношу — присылает менеджер.

### 4b Сроки / форма результатов теста
**Фотон:** 1) Результат сразу после теста — предварительный; открытые задания проверяются вручную. 2) После проверки менеджер направит итоговые результаты (как правило, в ближайшие дни). 3) По итогам распределяем по группам. Источник: [из диалогов — кандидат] + [KB knowledge_control].
**УНПК:** 1) После теста результаты смотрит куратор и распределяет по группам. 2) Итог сообщит менеджер; формируется группа по уровню. 3) Для смен — распределение по итогам тестирования перед сменой. Источник: [из диалогов — кандидат].
**Semantic-note:** бренд ок. Конкретные баллы и сроки — персональные/устаревшие, не выношу.

### 4c Деление по уровню → группа
**Фотон:** 1) По результатам теста распределяют по уровню: базовый / продвинутый (с элементами олимпиад) / олимпиадный. 2) Мест в группах ограничено, распределение по записи и итогам теста. 3) Уровень можно поменять, если легко/сложно. Источник: [KB levels_explained] + [из диалогов — кандидат].
**УНПК:** 1) Группы формируются по классу и уровню по результатам теста (базовые/профильные/олимпиадные). 2) На первом занятии может быть предложен переход в другую группу по уровню. 3) Перевод возможен по инициативе ученика или рекомендации педагога.
