# Telegram public pilot bots, 2026-05-21

## Что это

Локальный запуск двух Telegram-ботов:

- Фотон: `@foton_intellegence_bot`
- УНПК МФТИ: `@mipt_AI_bot`

Оба бота работают через long polling на MacBook. Webhook не нужен.

## Главные правила

1. Боты разделены по брендам:
   - бот Фотона отвечает только в рамках Фотона;
   - бот УНПК МФТИ отвечает только в рамках УНПК МФТИ.
2. База знаний берётся из актуального релиза:
   - `product_data/knowledge_base/kb_release_20260520_v6_3_team_answers/kb_release_v3_snapshot.json`
3. Ответ клиенту — только `draft_text`, без служебных полей, маршрутов, флагов, JSON и источников.
4. P0-защиты остаются в силе:
   - возврат денег;
   - жалобы;
   - юридические угрозы;
   - смешение брендов;
   - необоснованные обещания;
   - раскрытие внутренней природы системы.
5. В AMO, Tallanto и CRM runner ничего не пишет.
6. История диалога хранится только в памяти процесса. После перезапуска она очищается.
7. CRM/Tallanto подключены только в режиме чтения и только когда есть телефон из кодовой фразы.

## Кодовая фраза для сотрудников

Фраза:

```text

```

- включает тестовый режим для текущего Telegram-чата;
- добавляет телефон в контекст ответа;
- если телефон есть в локальной тестовой карте, добавляет имя родителя и ученика.
- запускает фоновое read-only подтягивание CRM/Tallanto-контекста.

Сейчас в локальной карте есть:

- телефон: `79092009933`
- родитель: `Ананьевская Анна Георгиевна`
- ученик: `Колосов Даниил Максимович`

Сейчас контекст работает так:

- локальная история по звонкам/контактам читается из текущего runtime-экспорта;

## CRM/Tallanto read-only

Режим задаётся в локальном файле:

```bash
MANGO_TELEGRAM_CRM_READ_MODE=server
MANGO_CRM_SERVER_URL=https://api.fotonai.online
MANGO_CRM_SERVER_API_KEY=<локальный ключ, не коммитить>
```

Возможные значения:

- `off` — не читать CRM/Tallanto;
- `local` — читать только локальный runtime-экспорт;
- `live` — читать локальный runtime-экспорт + live Tallanto + live AMO, если AMO подключён.
- `server` — читать локальный runtime-экспорт + AMO/Tallanto через `api.fotonai.online`.

Текущий результат проверки по телефону `79092009933`:

- локальный runtime: найден, 85 звонков;
- AMO через сервер: найдено 2 контакта и 4 сделки;
- Tallanto через сервер: найден 1 контакт;
- timeline hint: найден.

## Что проверять сотрудникам

1. Бренд не смешивается.
2. Возврат/жалоба/суд уходят в безопасную передачу менеджеру.
3. На зелёные темы бот отвечает полезно, а не только «уточню».
4. Если точного факта нет, бот задаёт 1-3 безопасных уточняющих вопроса и не выдумывает цену, дату, скидку или гарантию.
5. Кодовая фраза с номером включает тестовый режим.
6. `/reset` очищает тестовый режим и историю диалога.
