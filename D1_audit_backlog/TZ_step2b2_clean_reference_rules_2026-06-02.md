## Ключевые правила переноса (из реестра + контрактов)
1. **docs:** договор «в ближайшие дни»; справка «до 10 дней, постараемся раньше»; лицензия →
   «есть лицензия на образовательную деятельность» БЕЗ номеров/дат/юрлица. blocking → гейт:
   `pii_in_certificate_request` (ПДн не эхо), `legal_entity_details` (юрлицо/номер → менеджер).
2. **matkap:** федеральный да, региональный нет, одобрение — СФР (НЕ обещать). blocking → `sfr_approval_promise`.
3. **tax:** ФНС рассматривает и решает (НЕ гарантировать возврат); лицензия без номеров; справка помогает.
   blocking → `fns_return_guarantee`.
4. **olympiad:** Физтех онлайн ТОЛЬКО 9 и 11 классы; не подменять обычный курс; другой класс → менеджер
   сверит. blocking → `grade_outside_9_11_promise`.
5. **platform_access:** личный кабинет/как зайти/ссылка после оплаты/электронные документы — из факта.
   **ВАЖНО:** безопасная часть `_terminal_safe_template` (identity «ты бот», prompt-injection, off-topic,
   бренд-лояльность) НЕ мигрируется в правило — остаётся в гейте/identity-ветке (политика C уже покрыта
   тестом из 2b.1). Правило берёт ТОЛЬКО платформенный домен.
