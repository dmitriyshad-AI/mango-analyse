# ТЗ-18: гигиена гита эпохи пилота (без удаления файлов)

Дата: 2026-06-12. Постановщик: главный архитектор. Основание: ревизия состояния (ветка codex/tz14 в главной папке, 689 untracked, грязный CLAUDE.md, расхождение с main).

## Жёсткие рамки

1. НИЧЕГО не удалять (ни файлы, ни stash, ни ветки с уникальными коммитами). Кандидатов на удаление — только списком в отчёт (путь, размер, дата, почему не нужен) на решение Дмитрия.
2. Живые контуры работают из этой папки. Все операции со сменой ветки — в коротком окне: стоп-файл → пульс показывает остановку → git-работы → полный pytest → снять стоп → рестарт цикла и публичных ботов → `scripts/check_public_bot_live.py` + пульс + проверка журнала, что ретро-черновиков нет.
3. Никаких `git reset/checkout/clean` по живым изменениям; конфликты мержа решать в пользу свежих боевых коммитов, при сомнении — стоп и вопрос в отчёт.
