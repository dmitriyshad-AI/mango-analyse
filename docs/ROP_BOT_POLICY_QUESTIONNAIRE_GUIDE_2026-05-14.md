# Опросник для РОПа по правилам ответов бота

Дата: 2026-05-14

## Зачем нужен файл

Файл `product_data/question_catalog/rop_bot_policy_questionnaire_2026-05-14.xlsx` нужен не для проверки кода, а для принятия бизнес-решений:

- где бот может отвечать сам;
- где бот может отвечать только после проверки актуального факта;
- где бот только собирает данные и передает менеджеру;
- где отвечает только менеджер;
- какие темы нужно раздробить на более узкие случаи.

## Как заполнять

По каждой строке РОП смотрит:

1. `Блок для РОПа` и `Тема, которую нужно утвердить` — о чем эта строка.
2. `Как клиенты спрашивают` — реальные примеры обращений.
3. `Черновик безопасного ответа` — не финальный идеальный ответ, а безопасная заготовка.
4. `Рекомендация системы` — предварительное мнение системы, его можно менять.
5. `Вопросы 1-7` — что нужно решить.

РОП заполняет только колонки с началом `Ответ РОПа`.

## Что означают варианты разрешения боту

- `бот может ответить сам` — тема безопасная, ответ не требует персональной проверки.
- `бот может ответить только после проверки актуального факта` — нужна свежая цена, расписание, адрес, документ или карточка клиента.
- `бот только собирает данные и передает менеджеру` — бот не отвечает по сути, но может уточнить ФИО, класс, предмет, дату, сумму и т.д.
- `только менеджер` — тема рискованная: возвраты, договоры, спорные деньги, маткапитал, юридические вопросы.
- `тему нужно раздробить на более узкие случаи` — внутри одной строки смешаны разные ситуации, единый ответ утверждать нельзя.

## Что важно помнить

Исторический ответ менеджера из старых переписок не считается автоматически правильным. Менеджер мог знать контекст конкретного клиента, а бот должен работать безопасно и одинаково для похожих ситуаций.

Если РОП не уверен в теме, лучше выбрать более осторожный вариант: `бот только собирает данные и передает менеджеру` или `только менеджер`.

## Что получится после заполнения

После заполнения этого опросника можно будет собрать утвержденную базу правил:

- права самостоятельности бота;
- утвержденные формулировки;
- обязательные данные перед ответом;
- запреты и опасные обещания;
- список тем, которые нужно дробить дальше.
