# Implementation notes

Добавлен prompt-only санитар bot-safe памяти:

`_direct_path_bot_safe_memory_prompt_text()`

Он применяется только при рендере bot-safe памяти в prompt. Сами данные, SQLite, read_api и runtime builder не меняются.

Скрываются:

- `2025/26`;
- `12:15-14:15`;
- `12.06.2026`;
- `94 500 ₽`;
- `15%`.

Не скрывается смысловая нить: слова вроде «обсуждали расписание» остаются.

Числа из блока «Факты по вашему вопросу» не проходят через этот санитар.
