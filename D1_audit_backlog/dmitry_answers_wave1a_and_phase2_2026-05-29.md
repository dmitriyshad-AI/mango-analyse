### А.3 — Маркеры задержки для Спора 3: ПРИНЯТЫ

Триггер P0 — комбинация маркера задержки с темой документа:
- «уже месяц / уже неделю / уже X недель»
- «долго / так долго / так и не»
- «не могу получить»
- «всё ещё нет»
- В комбинации с: справку / документ / договор / доступ к ЛК / справку для вычета.

«Когда будет справка?» (без маркера задержки) — НЕ P0, нейтральный вопрос.

### А.4 — Маркеры качества для Спора 5: КОРРЕКЦИЯ

Дмитрий поправил мою гипотезу. «Непонятно объясняет» — **P0**.

Финальный список P0 для жалобы на преподавателя:
- «не нравится преподаватель / учитель»
- «плохо ведёт занятия»
- **«непонятно объясняет»** (поправка Дмитрия)
- «преподаватель часто опаздывает»
- «преподаватель пропускает занятия»
- «меняет преподавателя без предупреждения»

НЕ P0 (нейтральная обратная связь / уточнение):
- «слишком быстро говорит»
- «непонятно, можно ещё раз?» — это просьба разобрать, не претензия.

**Различие:** «непонятно ОБЪЯСНЯЕТ» (претензия к качеству) ≠ «непонятно, объясните ещё раз» (запрос
помощи).

### А.5 — Полный список P0-маркеров: ПРИНЯТ

Финальный список «реальных P0-маркеров» (бот молчит, передаёт менеджеру):

**1. Финансовая претензия СЕЙЧАС (с учётом намерения, не слов):**
- «верните деньги»
- «требую возврат»
- «отдайте деньги»
- «возвращайте» (только в контексте оплаты, не «возвращаете тетради»)

**2. Спор оплаты:**
- «двойное списание»
- «не списали»
- «провели платёж дважды»
- «деньги ушли — я не получил»
- «верните оплату»

**3. Юридическая угроза:**
- «обращаюсь в суд / прокуратуру / Роспотребнадзор»
- «жалоба»
- «обман / обманули»
- «мошенники»
- «введёте в заблуждение»

**4. Серьёзная жалоба на услугу:**
- «уже жалобу писал»
- Список А.4 (преподаватель)
- Список А.3 (задержка документа)
