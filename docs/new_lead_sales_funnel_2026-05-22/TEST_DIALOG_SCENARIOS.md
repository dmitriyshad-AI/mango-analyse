## Что превратить в автотесты

- P0 всегда не автономен.
- В УНПК нет `Т-Банк`, `Долями`, `через банк`.
- В Фотоне нет `УНПК`, `АНО ДПО`, `kmipt.ru`.
- В УНПК регулярная Москва не заменяется Красносельской.
- По лагерям бот спрашивает класс, а не возраст.
- Нет "места есть", "место закреплено", "оплата прошла".
- Нет "бесплатная пробная неделя" в УНПК.
- Нет "бесплатное очное пробное" без менеджера.
- Ответ содержит прямой ответ и один следующий шаг.
- Бот не спрашивает уже известные данные.
- Нет ФИО преподавателей, номеров лицензий, юрлиц и реквизитов без отдельного разрешения.
