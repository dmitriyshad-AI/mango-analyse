## Важное решение

`brand_neutral` теперь не используется для фактов с брендовыми словами, ценами, юрлицами, контактами, Т-Банком/Долями или рублёвыми условиями. Такие факты уходят во внутренний слой либо в отдельные брендовые записи.

Маткапитал и налоговый вычет есть для обоих брендов через отдельные факты `foton` и `unpk`; клиентский ответ всё равно фильтруется по активному бренду.
