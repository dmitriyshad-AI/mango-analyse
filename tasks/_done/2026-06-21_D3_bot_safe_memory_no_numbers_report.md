# D3: запрет чисел/дат/расписания из bot-safe памяти

Дата: 2026-06-21
Ветка: `codex/d3-phase01-botsafe-integration`
База: поверх `0d606c3` — гейт свежести памяти
Флаг: `TELEGRAM_BOT_SAFE_CRM_CONTEXT`, default OFF
Статус: реализация готова на ветке, в `main` не влита.

## Что изменено

В `_direct_path_bot_safe_context_prompt_block()` усилено правило работы с памятью:

- числа, даты, проценты, цены, расписание и адреса из bot-safe выжимки нельзя называть клиенту как факт;
- если такая деталь нужна, модель должна брать её только из блока «Факты по вашему вопросу»;
- если в «Фактах» детали нет, нельзя утверждать её из памяти, нужно предложить уточнить;
- память остаётся источником нити разговора: например, «обсуждали расписание», без точного времени/года/цены.

Дополнительно добавлен безопасный prompt-текст памяти:

`_direct_path_bot_safe_memory_prompt_text()`

Он скрывает точную конкретику в памяти до попадания в prompt:

- учебный год вида `2025/26`;
- время вида `12:15-14:15`;
- даты вида `12.06.2026`;
- цены и суммы вида `94 500 ₽`;
- проценты.

Блок «Факты по вашему вопросу» не трогается.

## Проверка на кейс 16

Контрольный memory-текст:

```text
Фотон: обсуждали расписание 2025/26, занятия 12:15-14:15, бюджет 94 500 ₽.
```

В prompt попадает:

```text
Фотон: обсуждали расписание <точная деталь из памяти скрыта>, занятия <точная деталь из памяти скрыта>, бюджет <точная деталь из памяти скрыта>.
```

Проверено:

- `2025/26` не попадает в prompt;
- `12:15-14:15` не попадает в prompt;
- `94 500` не попадает в prompt;
- фраза `обсуждали расписание` остаётся.

## Контроль: числа из «Фактов» не зарезаны

В том же prompt блок фактов содержит:

```text
Факт из базы: цена 47 250 ₽.
```

Проверено: `47 250` остаётся в prompt.

## NEG

- Флаг остался default OFF.
- На клиентах без bot-safe памяти поведение не меняется.
- Нить из памяти сохраняется, теряется только точная числовая конкретика.
- Блок «Факты по вашему вопросу» не фильтруется.
- Бренд-фильтр, PII-скан и гейт свежести не ослаблялись.
- Live-бот, AMO, Tallanto и боевая БД не трогались.
