- Содержание программы по классам и предметам (что именно проходят в 7 классе по математике, какие темы по неделям, какой учебник).

### Преподаватели
- ФИО и регалии преподавателей **вне Лобни и Менделеево** (Москва Сретенка, Москва Скорняжный, Долгопрудный Пацаева, Долгопрудный МФТИ).
- **Источник**: РОП.

### Расписание 2026/27
- Конкретные слоты групп (какая группа когда занимается).
- **Источник**: маркетинг. Будет к началу сентября 2026.

### Содержание интенсивов
- Какие именно темы изучаются на ОГЭ-интенсиве (Фотон) и ЕГЭ-интенсиве (УНПК) по неделям.

### Программы летних школ
- Подробное содержание ЛВШ и ЛШ городской — какие проекты, какие активности по дням.

### Отзывы и кейсы
- Истории успеха конкретных учеников («Иван поступил в МФТИ после нашего курса физики»).
- **Источник**: маркетинг.

### Внутренние процессы
- Регламенты «как менеджер оформляет договор пошагово», «как ведёт клиента по воронке».
- **Источник**: РОП.

### Контактные лица по узким темам
- Кто отвечает за лагерь (помимо Клычевой по маткапиталу).
- Кто отвечает за интенсивы.
- Кто отвечает за конкретный филиал.
- **Источник**: РОП.

### Программы УНПК 1-4 класс
- У УНПК есть курсы для 1-4 класса (цены подтверждены), но содержания программ нет.
- **Источник**: методист.

### Нестандартные кейсы по маткапиталу
- Несколько детей разных лет
- Ребёнок старше 24 лет
- Частичная оплата (маткапитал + деньги)
- **Источник**: Дарья Клычева (бухгалтерия).

### Нестандартные кейсы по налоговому вычету
- Онлайн-курс как очное обучение (позиция компании)
- За несколько лет одновременно
- **Источник**: бухгалтерия + юрист.

### Промокоды
- Все промокоды убраны из бота 2026-05-19 — у Дмитрия они «вообще удалены».
- В бот промокоды возвращаются только когда маркетинг даст список с `valid_until`.
- **Источник**: маркетинг (для будущего возврата в бота со сроками).

## Как использовать пробелы

Бот не должен делать вид, что знает то, чего нет. Если клиент спрашивает по теме из пробела:

«По этому вопросу подключу менеджера — он знает детали лучше меня.»

Менеджер, получая handoff, видит в системе пометку «бот не имел факта» и понимает, что клиент ждёт человека.
