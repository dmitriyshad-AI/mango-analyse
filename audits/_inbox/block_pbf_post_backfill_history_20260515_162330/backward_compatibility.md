# Backward Compatibility

## Сохраняется

- Хронология по двум и более содержательным звонкам сохраняется.
- `_history_line()` по-прежнему сжимает длинные строки без многоточий и с маркером `[сжато]`.
- `Краткая история общения` продолжает строиться для одного содержательного звонка.

## Меняется

- Для одного содержательного звонка поле `Хронология общения (последние 5 касаний)` теперь пустое.

Это намеренное изменение: оно убирает дублирование длинного пересказа последнего звонка.
